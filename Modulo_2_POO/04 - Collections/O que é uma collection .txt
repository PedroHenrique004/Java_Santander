Uma collection é uma estrutura de dados que serve para agrupar muitos elementos em uma unica unidade, esses elementos tem que ser objetos

Uma collection pode ter coleções homogeneas ou heterogeneas, normalmnente usamos coleções homogeneas

O nucleo das coleções é formado pelas interfaces da figura abaixo, essas interfaces permitem manipular a coleção independente do nível de detalhe que elas representem

---------------- 4 principais coleções ----------------------------

List -> lista

Set -> Conjunto

Queue -> fila 

Map -> mapa
